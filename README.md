#TraceParser - это консольный сканер на Python, предназначенный для автоматизированного анализа лог-файлов. Он помогает специалистам по кибербезопасности быстро находить следы активности в текстовых данных.

🚀 Основные возможности
Поиск IP-адресов: Автоматически извлекает все IPv4-адреса из файла.

Детекция хэшей: Находит строки, соответствующие формату SHA-256.

Режим Verbose: Возможность детального вывода найденных совпадений прямо в консоль в режиме реального времени.

Экспорт отчетов: Автоматическое создание файла report.txt с результатами сканирования для последующего анализа.

Обработка ошибок: Защита от сбоев при отсутствии файла или проблемах с кодировкой.

🛠️ Требования
Python 3.x

Библиотеки: re, argparse, sys (встроены в Python).


💻 Использование
Для запуска сканера используйте командную строку:

Bash

py TraceParser.py --file <путь_к_вашему_логу> --verbose

Параметры:
--file: (Обязательно) Путь к файлу логов, который нужно проанализировать.

--verbose: (Опционально) Включает подробный вывод информации о каждой находке в консоль.
